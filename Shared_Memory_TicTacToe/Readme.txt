A text based version of TicTacToe.
This program uses two different processes to serve as player 1 and player 2
To connect the two different processes together they use shared memory
The game can be played on board sizes from 3-20
Player 1 serves as the main program printing out the game board to the console
Both Player 1 and Player 2 enter their moves on their own process


To run this program, first run p1.cpp and then run p2.cpp
Uses Linux dependancys so must be run on a Linux or Unix System, Virtual Machine, or Docker Container
